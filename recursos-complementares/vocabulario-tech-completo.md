## 🗣️ FRASES PRONTAS PARA CONVERSAÇÃO

### **👋 APRESENTAÇÃO PROFISSIONAL**
```
"Hi, I'm [name]. I'm a software engineer specializing in [area]."
"I have [X] years of experience in [technology stack]."
"Currently, I work for [company] developing [type of applications]."
"My main focus is [frontend/backend/full-stack] development."
"I'm passionate about [technology/area] and always learning new things."
```

### **💼 FALANDO SOBRE PROJETOS**
```
"I'm currently working on [project description]."
"The project involves [technologies] and [main functionality]."
"We're building [type of application] using [tech stack]."
"The biggest challenge is [specific challenge]."
"I implemented [feature] using [technology/approach]."
"The system handles [number] users/requests per day."
```

### **🛠️ DISCUSSÕES TÉCNICAS**
```
"We use [technology] for [purpose]."
"The architecture follows [pattern/methodology]."
"I prefer [technology A] over [technology B] because [reason]."
"Have you worked with [technology] before?"
"What's your experience with [framework/tool]?"
"How do you usually handle [technical challenge]?"
```

### **🐛 RESOLUÇÃO DE PROBLEMAS**
```
"I'm debugging an issue with [component/system]."
"The error message says [error description]."
"I think the problem is in the [part of system]."
"Let me check the logs to see what's happening."
"I found the root cause - it's [explanation]."
"The fix involves [solution approach]."
```

### **📈 PERFORMANCE E OTIMIZAÇÃO**
```
"The application is running slowly."
"We need to optimize [specific part]."
"I implemented caching to improve performance."
"The database queries were taking too long."
"We reduced loading time by [percentage/amount]."
"The new version is [X]% faster."
```

## 🎯 EXERCÍCIOS PRÁTICOS

### **📝 EXERCÍCIO 1: ELEVATOR PITCH TÉCNICO (90 segundos)**
Complete e pratique:
```
"Hi! I'm [your name], a [your role] with [X] years of experience.
I specialize in [your main technologies].
Currently, I'm working on [current project] using [technologies].
What I love about programming is [personal motivation].
I'm always interested in [learning area/new technology].
What kind of projects are you working on?"
```

### **🎤 EXERCÍCIO 2: DESCRIÇÃO DE PROJETO (2 minutos)**  
Escolha um projeto seu e pratique esta estrutura:
```
"Let me tell you about [project name].
It's a [type of application] that [main purpose].
We built it using [technologies].
The main features include [list 3-4 features].
The biggest technical challenge was [challenge].
We solved it by [solution].
The result was [outcome/impact]."
```

## 💡 DICAS DE PRONUNCIA E CONVERSAÇÃO

### **🗣️ PALAVRAS COMUMENTE MALPRONUNCIADAS**
```
❌ ERRADO → ✅ CORRETO
• "Java" como "javá" → /ˈdʒɑːvə/ (JÁ-va)
• "Cache" como "caché" → /kæʃ/ (CASH)
• "Route" como "rout" → /ruːt/ (ROOT)
• "SQL" como "sequel" → /ˈɛskjuːˈɛl/ (ES-QUE-EL)
• "GitHub" → /ˈɡɪt-hʌb/ (GIT-hub, não "git-rub")
• "Linux" → /ˈlɪnəks/ (LI-nux)
• "NGINX" → /ˌɛnˈdʒɪnks/ (en-jin-X)
```

### **💬 CONECTORES PARA CONVERSAS TÉCNICAS**
```
• "Speaking of [topic]..." - Para mudar de assunto
• "That reminds me of..." - Para adicionar informação relacionada
• "In my experience..." - Para compartilhar experiência
• "From a technical perspective..." - Para análise técnica
• "The way I see it..." - Para dar opinião
• "On the other hand..." - Para contrastar
• "That's a good point, but..." - Para educadamente discordar
```

### **🤝 EXPRESSÕES PARA NETWORKING**
```
• "What stack do you usually work with?"
• "Have you tried [technology] before?"
• "I'd love to hear about your experience with [topic]."
• "That sounds really interesting! Tell me more."
• "We should definitely stay in touch."
• "Feel free to reach out if you have any questions."
• "I'd be happy to share my experience with [technology]."
```
